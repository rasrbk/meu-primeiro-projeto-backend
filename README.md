Projeto de desenvolvimento de uma API como projeto prático do curso “Minha Primeira API” da PrograMaria, utilizando JavaScript, Node.js e MongoDB. O projeto envolve a criação de um servidor e a implementação de operações CRUD (GET, POST, PATCH e DELETE), aplicando conceitos fundamentais de back-end e manipulação de dados. A experiência contribui para a construção de uma base sólida em desenvolvimento web e para a criação de um portfólio inicial na área de tecnologia.
